ðŸ”¬ WHAT IS GUTT?
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

**GUTT = Generative Unit Task Template**
- Framework for decomposing user prompts into atomic unit tasks
- Each GUTT represents a single capability/skill
- Used to evaluate AI system's ability to break down complex requests

**ACRUE Quality Dimensions:**
- A = Accurate (factual correctness)
- C = Complete (comprehensive coverage)
- R = Relevant (context-appropriate)
- U = Useful (goal achievement)
- E = Exceptional (competitive advantage)

**Scoring:** 1-4 scale per dimension
- 4 = Exceptional
- 3 = Strong
- 2 = Adequate
- 1 = Poor
